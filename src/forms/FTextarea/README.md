# FTextarea

`FTextarea` is a textarea component with built-in styling, validation states, and size variants.

### Usage

Import the component:

```js
import { FTextarea } from 'futility-ui';
// or
import FTextarea from 'futility-ui/forms/FTextarea';
```

Use it in your template:

```html
<FTextarea v-model="value" placeholder="Write text here ..." />
```

That's it!

# FNumberPicker

`FNumberPicker` is a numeric input with increment and decrement buttons.

### Usage

Import the component:

```js
import { FNumberPicker } from 'futility-ui';
// or
import FNumberPicker from 'futility-ui/forms/FNumberPicker';
```

Use it in your template:

```html
<FNumberPicker v-model="quantity" :min="0" :max="99" />
```

That's it!

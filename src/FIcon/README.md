# Futility UI / Icons

`FIcon` is the shared icon component used across the library. The source icon set is available on [Flowbite Icons](https://flowbite.com/icons/). Icons are being migrated into the library incrementally.


### Usage

Import the component:

```js
import { FIcon } from 'futility-ui';
// or
import FIcon from 'futility-ui/FIcon';
```

Use it in your template:

```html
<FIcon name="archive_solid" />
```

That's it!
